Игра "Быки и Коровы"

Простая консольная игра на C++.

* Как скомпилировать и запустить:
    1. Откройте терминал в папке с файлами
    2. Введите команду: g++ main.cpp game_logic.cpp file_operations.cpp interface.cpp -o bulls_and_cows.exe
    3. Запустите игру: bulls_and_cows.exe

* Что делает игра:
    - Компьютер загадывает число
    - Вы пытаетесь его угадать
    - После каждой попытки показывает:
    - "Быков" (цифра на своем месте)
    - "Коров" (цифра есть, но не на своем месте)
    - При игре вдвоём один игрок загадывает число другому, правила не меняются

* Файлы в проекте:
    - "bulls_and_cows.h" - заголовочный файл
    - "main.cpp" - главное меню
    - "game_logic.cpp" - генерация числа и проверки
    - "interface.cpp" - меню и интерфейс
    - "file_operations.cpp" - сохранение рекордов

!При возникновении проблем с русским языком нужно сменить кодировку!
